Instagram User ID & Token Aracı

Hesap: @reci3d · Instagram API with Instagram Login (2026)
📋 Token nasıl alınır? (adım adım)
  1. Instagram uygulamasında: Ayarlar → Hesap türü ve araçlar → Profesyonel hesaba geç → Creator (ücretsiz, şart).
  2. developers.facebook.com/apps → Create App → tür olarak Business seç.
  3. App panelinde sol menü: Instagram → API setup with Instagram business login.
  4. 1. Generate access tokens bölümünde Add account ile @reci3d'yi bağla → Generate Token.
  5. Açılan Instagram penceresinde izin ver → çıkan 60 günlük token'ı kopyala.
  6. Token'ı aşağıya yapıştır → Bilgileri Getir.
